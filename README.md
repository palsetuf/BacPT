# BacPT
**Bacterial Proteome Foundation Model for Enhanced Functional Prediction**

Code for the manuscript: _Bacterial proteome foundation model enhances functional prediction from enzymes to ecological interactions_

---

## Repository Structure
- **training/** - Model training scripts
- **inference/** - Model loading and inference utilities  
- **notebooks/** - Analysis notebooks
- **data/** - Sample data
- **environment.yml** - Conda environment specification

---

## Installation

```bash
conda env create -f environment.yml
conda activate bacpt
```
